Tabla en hmtl Tabla en HTML: Una tabla en HTML es una estructura utilizada para organizar datos en filas y columnas. Se emplea para mostrar información de manera clara y estructurada, facilitando la comparación de datos. Formulario en html Formulario en HTML: Un formulario en HTML es una sección de una página web que contiene controles interactivos que el usuario puede utilizar para enviar datos al servidor web. Los formularios se crean usando la etiqueta (form) y pueden incluir varios elementos de entrada como campos de texto, botones, menús desplegables, casillas de verificación, y más.
Etiqueta table Etiqueta (table): La etiqueta (table) es el contenedor principal de una tabla en HTML. Define el inicio y el fin de una tabla. Etiqueta from Etiqueta (form): La etiqueta (form) define un formulario HTML. Esta etiqueta actúa como un contenedor para todos los elementos de entrada del formulario, como campos de texto, botones, etc. También incluye atributos importantes como action (URL donde se envían los datos) y method (método HTTP utilizado para enviar los datos, como GET o POST).
Atributos de table Atributos de (table): border: Define el grosor del borde de la tabla. cellpadding: Define el espacio entre el borde de una celda y su contenido. cellspacing: Define el espacio entre las celdas de la tabla. width: Define el ancho de la tabla. height: Define la altura de la tabla. align: Define la alineación de la tabla en la página (left, center, right). Etiqueta input Etiqueta (input): La etiqueta (input) se utiliza para crear varios tipos de controles de entrada en un formulario. Los tipos más comunes incluyen text, password, radio, checkbox, submit, y button, determinados por el atributo type.
Etiqueta tr Etiqueta (tr): La etiqueta (tr) define una fila en una tabla HTML. Dentro de (tr) se colocan las celdas de la fila, que se definen con las etiquetas (td) o (th). Etiqueta label Etiqueta (label): La etiqueta (label) se utiliza para definir etiquetas para los elementos de entrada en un formulario. Mejora la accesibilidad y la usabilidad al asociar una etiqueta con un elemento de entrada. Se puede asociar un (label) con un control de formulario usando el atributo for, que debe coincidir con el atributo id del control.
Atributo tr Atributos de (tr): align: Define la alineación horizontal del contenido en las celdas de la fila. bgcolor: Define el color de fondo de la fila. Etiqueta select Etiqueta (select): La etiqueta (select) crea un menú desplegable. Dentro de (select), se utilizan etiquetas (option) para definir las opciones que el usuario puede seleccionar. Atributos importantes incluyen name y multiple (para permitir seleccionar múltiples opciones).
Etiqueta td y th Etiquetas (td) y (th): (td): Define una celda de datos en una tabla. (th): Define una celda de encabezado en una tabla. El contenido dentro de (th) se muestra en negrita y centrado de manera predeterminada. Etiqueta textarea Etiqueta (textarea): La etiqueta (textarea) define un área de texto grande y de múltiples líneas en un formulario, ideal para comentarios u opiniones. Atributos importantes incluyen rows y cols, que especifican el tamaño del área de texto.
Atributos de td y th Atributos de (td) y (th): colspan: Define cuántas columnas abarca la celda. rowspan: Define cuántas filas abarca la celda. align: Define la alineación horizontal del contenido en la celda. valign: Define la alineación vertical del contenido en la celda. bgcolor: Define el color de fondo de la celda. Etiqueta fieldset y legend Etiquetas (fieldset) y (legend): (fieldset): Agrupa elementos relacionados en un formulario, proporcionando un borde alrededor de esos elementos. (legend): Proporciona un título o leyenda para un grupo de elementos definidos por (fieldset).
Tablas avanzadas Tablas avanzadas: Las tablas avanzadas en HTML incluyen el uso de las etiquetas (thead), (tbody), y (tfoot) para separar claramente el encabezado, el cuerpo y el pie de la tabla, permitiendo una mejor organización y estilo. Etiqueta meter Etiqueta (meter): La etiqueta (meter) se utiliza para representar un valor en un rango conocido, como un medidor de combustible o una barra de progreso. Atributos importantes incluyen min, max, y value.
Etiqueta thead Etiqueta (thead): La etiqueta (thead) se utiliza para agrupar el conjunto de filas que forman el encabezado de la tabla. Normalmente contiene una o varias etiquetas (tr) con etiquetas (th) Etiqueta progress Etiqueta (progress): La etiqueta (progress) representa el progreso de una tarea. Similar a (meter), pero específicamente para mostrar progreso. Atributos importantes incluyen max y value.
Etiqueta tfoot Etiqueta (tfoot): La etiqueta (tfoot) se utiliza para agrupar el conjunto de filas que forman el pie de la tabla. Se coloca normalmente después de (thead) y (tbody). Etiqueta datalist Etiqueta (datalist): La etiqueta (datalist) proporciona una lista de opciones predefinidas para un elemento (input). El usuario puede seleccionar una de las opciones o ingresar un valor diferente.
Etiqueta tbody Etiqueta (tbody): La etiqueta (tbody) se utiliza para agrupar el conjunto de filas que forman el cuerpo de la tabla. Normalmente contiene varias etiquetas (tr) con etiquetas (td). Etiqueta summary Etiqueta (summary): La etiqueta (summary) se utiliza como un encabezado visible para un elemento (details), que puede contener información adicional o detalles colapsables que se muestran o se ocultan al hacer clic.

1. Contenedor (weapper)

Contenedor (wrapper): El contenedor es un elemento que agrupa y envuelve todos los elementos de la página web, proporcionando una estructura básica y facilitando la organización y el estilo del contenido. Suele utilizar la etiqueta (div) con una clase o id específico.

2. Cabecera (header)

Cabecera (header): La cabecera es la parte superior de la página web que generalmente contiene el logo, el título del sitio, y el menú de navegación principal. Se suele definir con la etiqueta (header)

3. Contenido (content)

Contenido (content): El contenido es la sección principal de la página web donde se encuentra la información relevante que se desea comunicar a los visitantes. Puede incluir texto, imágenes, videos y otros elementos multimedia.

4. Menú (menu)

Menú (menu): El menú es una lista de enlaces de navegación que permite a los usuarios acceder a las diferentes secciones o páginas del sitio web. Se puede crear utilizando una lista no ordenada (ul) y estilizada con CSS.

5. Pie (footer)

Pie (footer): El pie es la parte inferior de la página web que típicamente contiene información de contacto, enlaces a políticas de privacidad y términos de uso, así como otros enlaces secundarios. Se define con la etiqueta (footer).

6. Lateral (sidebar)

Lateral (sidebar): La barra lateral es una columna adicional situada a la derecha o izquierda del contenido principal, que puede contener enlaces adicionales, menús, widgets, publicidad o cualquier otro contenido complementario.

7. Etiqueta (div)

Etiqueta (div): La etiqueta (div) es un contenedor genérico en HTML que se utiliza para agrupar elementos y aplicar estilos a un grupo específico de elementos a través de CSS. Es muy versátil y común en la estructura de páginas web.

La estructuración semántica de una página web se refiere a la utilización de etiquetas HTML que no solo estructuran el contenido, sino que también proporcionan un significado claro sobre el tipo de contenido que contienen. Esto mejora tanto la accesibilidad como el SEO (optimización para motores de búsqueda).
